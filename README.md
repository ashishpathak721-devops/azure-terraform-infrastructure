# azure-terraform-infrastructure
Azure Infrastructure deployment using Terraform including Bastion, Windows/Linux VMs, VNet Peering, Load Balancer and NGINX Web Application.

## Components 
- Azure Bastion
- Windows VM
- Linux VM
- VNet Peering
- Azure Load Balancer
- Backend Pool
- Health Probe
- LB Rule
- NGINX Web Server
  
## Terraform Concepts
- for_each
- Nested Maps
- Variables
- Resource Dependencies

## Validation
- Bastion Login Successful
- Windows to Linux SSH Successful
- NGINX Deployment Successful
- Load Balancer Validation Successful
